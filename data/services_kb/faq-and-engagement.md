# FAQ and engagement details

## How does pricing work?

Fixed scope, fixed price, agreed up front after a short discovery call — not open-ended hourly billing. Typical bands: website support agent $300–800, lead-capture/booking agent $400–1,000, workflow automation $200–600, agentic function (sales/email role) $500–1,200, internal-tooling integration $400–1,000, reliability audit $500+. Entry pricing is on the lower end while I build up reviews; it climbs with track record.

## How long does it take?

A focused single agent is usually 1–2 weeks part-time. Workflow automations can be a few days. Multi-step agentic roles and internal integrations run 2–3 weeks depending on how many systems they touch. See the approach-and-process doc for the phase breakdown.

## Which LLM do you use? Am I locked in?

You're not locked in. The agent is provider-agnostic — Anthropic (Claude), OpenAI, or Google Gemini, selectable at runtime — because every vendor sits behind one interface. We pick the model that fits your accuracy and cost needs, and can switch later without a rewrite.

## What will it cost me to run?

Every run is traced with token counts and a dollar estimate, and there's a hard daily cost ceiling plus rate-limiting so a bot or bored visitor can't run up the bill. You'll see cost-per-conversation in a small dashboard. For most small-business volumes this is a few dollars a month, not a scary number.

## What do you need from me?

Your docs/content, access (or a sandbox) for any tools the agent integrates with, and a contact for the discovery call.

## Can you prove it works before I commit?

Yes — that's the whole point of this site. The Nimbus demo is a working agent I evaluated at 92.7% accuracy, and the page you're reading is itself running my agent. I bring the same evaluation report to your project.

## Do you host it, or do I?

Either. I can deploy it (Railway/Fly/Render for the backend, Vercel for a site) or hand it over to run on your infrastructure with the embeddable widget. Internal deployments stay behind your authentication.
